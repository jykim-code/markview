파일 URL 액세스 권한이 필요합니다

로컬 .md 파일을 Markview에서 열려면 확장 프로그램에 파일 URL 액세스 권한을 허용해야 합니다.

  1. 주소창에 chrome://extensions (Edge: edge://extensions) 를 입력하세요
  2. Markview - Markdown Viewer 확장 프로그램의 세부정보를 클릭하세요
  3. 파일 URL에 대한 액세스 허용 토글을 켜세요
  4. 이 탭을 닫고, .md 파일을 다시 열어보세요

설정 후에도 이 페이지가 보인다면, 브라우저를 완전히 종료 후 다시 시작해보세요.
